# Инструкция по тестированию приложения через FastAPI UI

**Так как задеплоить это упражнение - ЭТО ТО ЕЩЕ УПРАЖНЕНИЕ, а во время локального тестирования я чувствовал себе сапером (ох уж этот FastAPI - и это я еще писал сам сервис!), то однозначно нужна инструкция по поводу того, как это все тестировать**

## Шаг 1 — Запуск приложения

Открой терминал, перейди в папку проекта и выполни:

docker compose up --build

Подожди, пока в терминале появится строка:

INFO: Uvicorn running on http://0.0.0.0:8000

Это значит, что приложение запущено, и можно начинать тонуть в этом... Я имел в виду тестировать сервис.

## Шаг 2 — Открыть интерфейс

Перейди по адресу: http://localhost:8000/docs

## Шаг 3 — Регистрация

1. Найди раздел **auth** и нажми на `POST /auth/register`
2. Нажми кнопку **Try it out**
3. В поле **Request body** заполни:
{
  "username": "ваш_ник",
  "email": "ваш@email.com",
  "password": "ваш_пароль"
}
4. Нажми **Execute**
5. В ответе должен прийти `201 Created` с данными пользователя

## Шаг 4 — Вход и получение токена

1. Нажми на `POST /auth/login`
2. Нажми **Try it out**
3. Заполни:
{
  "username": "ваш_ник",
  "password": "ваш_пароль"
}
4. Нажми **Execute**
5. В ответе найди поле `access_token` и **скопируй его значение** — оно понадобится на следующем шаге

## Шаг 5 — Авторизация в интерфейсе

Без этого шага все защищённые эндпоинты будут возвращать ошибку 401.

1. Нажми кнопку **Authorize** в правом верхнем углу страницы (значок 🔓)
2. В поле **Value** вставь токен как есть (БЕЗ ПРОБЕЛОВ, ЛИШНИХ СЛОВ И СИМВОЛОВ):
3. Нажми **Authorize**, затем **Close**

Теперь все запросы будут отправляться с твоим токеном автоматически.

**ВАЖНО: после этого нельзя перезагружать страницу - иначе придется авторизовываться заново через логин**

## Шаг 6 — Проверка своего профиля

1. Нажми `GET /auth/me`
2. **Try it out** → **Execute**
3. Должны вернуться данные твоего пользователя

Если все вернулось корректно - значит это прекрасно, и все было сделано правильно)

## Шаг 7 — Установка любимого слова (бонусная фича)

1. Нажми `POST /auth/me/favorite-word`
2. **Try it out**
3. Заполни:
{
  "word": "govnokod"
}
4. **Execute** — слово сохранится в профиле
Чтобы удалить любимое слово: `DELETE /auth/me/favorite-word` → **Try it out** → **Execute**

## Шаг 8 — Сокращение ссылки (то, ради чего мы все здесь собрались и к чему мы уже час как не может подойти)

1. Найди раздел **links**, нажми `POST /links/shorten`
2. **Try it out**
3. Минимальный вариант:
{
  "original_url": "https://www.google.com"
}
4. Расширенный вариант (со своим алиасом и сроком действия):
{
  "original_url": "https://www.google.com",
  "custom_alias": "mozillafirefox",
  "expires_at": "2027-01-01T00:00:00"
}
5. **Execute** — в ответе придёт `short_code`, он понадобится для следующих шагов

**ВАЖНО: этот json по структуре более хрупкий чем мой заказ на озоне, поэтому тут нельзя ставить лишние запятые - иначе выпадет ошибка**

## Шаг 9 — Переход по короткой ссылке

Просто вставь в адресную строку браузера:
http://localhost:8000/links/ВАШ_SHORT_CODE
Браузер должен перенаправить тебя на оригинальный URL. Каждый переход увеличивает счётчик кликов.

## Шаг 10 — Статистика ссылки

1. Нажми `GET /links/{short_code}/stats`
2. **Try it out**
3. В поле `short_code` введи свой код
4. **Execute** — увидишь количество кликов, дату создания, дату последнего перехода

## Шаг 11 — QR-код для ссылки

1. Нажми `GET /links/{short_code}/qr`
2. **Try it out**
3. Введи `short_code`
4. **Execute**
5. Нажми на ссылку в поле **Response body** чтобы скачать и открыть картинку с QR-кодом

## Шаг 12 — Поиск ссылки по оригинальному URL

1. Нажми `GET /links/search`
2. **Try it out**
3. В поле `original_url` введи полный адрес, например `https://www.google.com`
4. **Execute** — найдёт короткую ссылку для этого URL

## Шаг 13 — Обновление ссылки

Изменить куда ведёт ссылка или переименовать её код (только для своих ссылок):

1. Нажми `PUT /links/{short_code}`
2. **Try it out**
3. Введи текущий `short_code` и заполни тело запроса:
{
  "original_url": "https://www.youtube.com",
  "new_short_code": "mytube"
}
4. **Execute**

## Шаг 14 — Удаление ссылки

1. Нажми `DELETE /links/{short_code}`
2. **Try it out**
3. Введи `short_code`
4. **Execute** — вернётся `204 No Content`, ссылка удалена

## Шаг 15 — Бонусные эндпоинты

Найди раздел **extras**:

- `GET /extras/joke` — случайный анекдот из коллекции автора
- `GET /extras/self-promotion` — ссылка на Telegram-канал автора
- `GET /extras/secret-url` — секретный URL (попробуй сам 👀)

Для всех трёх: **Try it out** → **Execute**.

## Остановка приложения

Когда закончишь тестирование, в терминале выполни:

docker compose down

Можно клassический ctrl-c, а потом выйти и забыть...
